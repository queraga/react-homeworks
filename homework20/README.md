# User List (Redux Toolkit + useSelector)

A small React practice project that displays a list of users from Redux state using `useSelector`.

## Features

- Redux Toolkit store setup
- `userSlice` with initial users data
- `UserList` component reads users via `useSelector`
- Renders user **name** and **email** in a list

## Tech Stack

- React (Vite)
- Redux Toolkit
- React Redux
